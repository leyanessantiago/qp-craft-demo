# Craft Demo

## Description

It uses React + TypeScript + Vite.

## Setup

```sh
nvm use

# NOTE: should use node v18.18.2 (npm v9.8.1)
```

```sh
npm install

# install node modules
```

### To start the project in production mode

```sh
npm run build

npm run preview
```

### For development

```sh
npm run dev
```
